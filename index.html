CREAR ARCHIVO

Dentro de VS, ir a crear carpeta, colocar nombre que se le desea y colocar al final la extension ".html" Ir a file, new file o usar atajo CTRL+N, crea una carpeta y se guarda en SAVE o SAVE AS y se le pone el nombre más la extension LA PRIMER LINEA DE UN ARCHIVO HTML ES

ELEMENTOS Y ETIQUETAS (Elements & Tags)

Componente basico de un html, las etiquetas definen elementos en el HTML ETIQUETA APERTURA: ETIQUETA DE CIERRE: >/html> ej: Algunos elementos no necesitan elementos de cierre (self-closingtags) como o ej --> elemento raiz EL CONTENIDO VA DENTRO DEL ELEMENTO RAIZ --> elemento raiz luego: --> cabeza Elementos importantes para la pagina pero no figuran en la visualizacion ej: enlace a carpeta.css, etc
